01

Claude Code 是什麼

Claude Code 是在終端機（PowerShell）中運作的 AI 助理工具。可以想像成一位坐在您電腦前的助手：您用白話下指令，它直接動手操作您的檔案。

Claude 網頁版Claude Code
檔案存取手動上傳、複製貼上直接讀寫電腦上的檔案
處理量一次一份可批次處理整個資料夾
執行動作只能給文字建議可執行指令、修改檔案、產出成品
安全機制—每個動作前先徵求您同意
帳號需求需 Claude Pro 以上付費方案，或 Anthropic API 金鑰。免費方案不含 Claude Code 使用權限。
02

安裝（Windows 原生版）

STEP 1

執行安裝指令

按 Win 鍵 → 輸入 powershell → 開啟（不需系統管理員身分），貼上以下指令後按 Enter：

Windows PowerShell
PS C:\Users\you> irm https://claude.ai/install.ps1 | iex
注意貼上後畫面可能停頓數分鐘，看起來像當機——這是正常現象，請耐心等待，不要關閉視窗。
STEP 2

處理 PATH 環境變數（最常見卡關點）

若安裝訊息提示 .local\bin is not in your PATH，或之後輸入 claude 顯示「不是內部或外部命令」，執行以下指令一次解決：

Windows PowerShell
PS C:\Users\you> [Environment]::SetEnvironmentVariable("Path", $env:Path + ";C:\Users\$env:USERNAME\.local\bin", "User")

執行後關閉所有 PowerShell 視窗並重新開啟，設定才會生效。

STEP 3

驗證安裝

Windows PowerShell
PS C:\Users\you> claude --version
成功標誌顯示版本號（例如 2.x.x）即代表安裝完成。
建議加裝

Git for Windows

非必要但強烈建議。未安裝時 Claude Code 以 PowerShell 執行指令；安裝後可使用功能較完整的 Bash 工具。

  • 下載：git-scm.com/downloads/win
  • 安裝時全程使用預設選項即可
03

首次啟動

先建立一個獨立的練習資料夾，再進入該資料夾啟動：

Windows PowerShell
# 建立並進入練習資料夾 PS C:\Users\you> mkdir $HOME\Desktop\claude-test PS C:\Users\you> cd $HOME\Desktop\claude-test # 啟動 Claude Code（首次會引導登入 Claude 帳號） PS ...\claude-test> claude
重要安全原則啟動時會詢問是否「信任此資料夾（Trust folder）」。切勿在家目錄（C:\Users\您的帳號）直接啟動並信任——Claude Code 會取得該目錄下所有檔案的讀寫權限。務必先建立獨立工作資料夾，再進入啟動。

啟動成功後，直接用白話輸入第一個任務試試看：

Claude Code
> 請建立一個 hello.txt，內容寫「我的第一個 Claude Code 任務成功了！」

它會詢問是否允許建立檔案，按 y 同意。打開資料夾看到 hello.txt，就代表一切正常。

04

三個核心觀念

信任資料夾

Claude Code 只能操作您信任的資料夾範圍。一個專案一個資料夾，範圍愈小愈安全。

權限確認

每次寫入檔案或執行指令前，畫面會跳出確認，按 y 同意。這是安全機制，不是故障。

CLAUDE.md 工作守則

輸入 /init 產生守則檔。寫入固定要求（如：一律使用繁體中文），之後每次對話自動遵守。

05

常用指令速查

指令／按鍵功能
/help查看所有可用指令
/init建立 CLAUDE.md 工作守則檔
/clear清除目前對話，重新開始
Esc中斷正在執行的動作
Ctrl + C（按兩次）離開 Claude Code
claude --version（於 PowerShell）查看版本
06

實用範例：不會寫程式也能做的事

第一個真實任務建議從低風險、結果看得到的工作開始。以下為適合新手的起步範例：

批次整理檔案
「把這個資料夾裡的照片，依拍攝日期分類，各建子資料夾歸檔，並產出一份清單。」
批次重新命名
「將資料夾裡 50 個檔案，統一改名為『2026-月份-主題』格式。」
彙整多份文件
「這裡有 20 份 Word 會議記錄，擷取每份的結論與待辦事項，彙整成一張 Excel 總表。」
製作個人網頁
「幫我做一個個人作品集網頁，深色風格，包含自我介紹與三個作品區塊。」
修改現有網頁
「幫我在這個網頁加一個匯出 CSV 的按鈕。」——它會自行讀檔、改碼、存檔。
操作原則用白話描述目標即可，不需懂程式。結果不對就直接說「不對，改成⋯⋯」，它會修正重做。重要檔案請先備份一份再交給它處理。
07

常見問題

輸入 claude 顯示「不是內部或外部命令」？

PATH 環境變數未設定。回到第二節 STEP 2 執行設定指令，並記得關閉所有 PowerShell 視窗後重新開啟。

瀏覽器登入後，終端機沒有反應？

稍等幾秒，認證資訊需要時間回傳。若超過 30 秒仍無反應，按 Ctrl + C 取消後重新執行 claude。

它一直跳出確認訊息，很煩，可以關掉嗎？

權限確認是核心安全機制，建議新手保留。熟悉後可在對話中選擇「本次工作階段皆允許」減少詢問次數，但不建議完全關閉。

會不會把我的檔案弄壞？

它只能操作您信任的資料夾，且每個動作都先徵求同意。養成兩個習慣即可安心使用：重要檔案先備份、一個專案用一個獨立資料夾。

需要付費嗎？

需要 Claude Pro（含）以上的付費方案，或使用 Anthropic API 金鑰按用量計費。免費帳號無法使用 Claude Code。

想深入學習，去哪裡看？

官方文件：docs.claude.com/en/docs/claude-code/overview，或在 Claude Code 內輸入 /help。
